# House of Diamonds Landing Page - Pre-Launch Checklist

## 1. Google Ads Setup

### Conversion Tracking
- [ ] Google Ads conversion tracking pixel installed on landing page
- [ ] Conversion tracking pixel installed on thank-you/confirmation page
- [ ] Test conversion tracking by submitting a test form
- [ ] Verify conversion appears in Google Ads within 24 hours
- [ ] Set up conversion value (if applicable)
- [ ] Configure conversion window (30 days recommended)
- [ ] Enable view-through conversions
- [ ] Link Google Ads account to Google Analytics 4

### Campaign Structure
- [ ] Create campaign with appropriate daily budget
- [ ] Set up ad groups by jewelry category (engagement rings, wedding bands, etc.)
- [ ] Write compelling ad copy with unique selling propositions
- [ ] Include relevant keywords (long-tail and short-tail)
- [ ] Set up negative keywords list
- [ ] Configure ad extensions (sitelinks, callouts, structured snippets)
- [ ] Add call extension with phone number
- [ ] Set up location targeting (Sydney-based or specified areas)
- [ ] Configure ad schedule (if needed)
- [ ] Set up bid strategy (Manual CPC or Target CPA)
- [ ] Add call-to-action extensions
- [ ] Review quality score factors
- [ ] Set up remarketing audiences

### Landing Page Quality
- [ ] Ensure landing page URL matches ad destination URL
- [ ] Verify page loads in under 3 seconds
- [ ] Confirm mobile-friendly design
- [ ] Check that headline matches ad promise
- [ ] Verify clear call-to-action above the fold

## 2. Landing Page Optimizations

### Meta Tags & SEO
- [ ] Title tag is compelling and under 60 characters
- [ ] Meta description is persuasive and under 160 characters
- [ ] Open Graph tags configured for social sharing
- [ ] Twitter Card tags configured
- [ ] Canonical URL tag set
- [ ] Robots meta tag configured correctly
- [ ] Schema markup added (LocalBusiness, Organization)
- [ ] Favicon uploaded and displaying correctly
- [ ] Alt text added to all images
- [ ] H1 tag is clear and contains primary keyword

### Images & Media
- [ ] All images optimized and compressed (WebP format where possible)
- [ ] Images under 200KB each
- [ ] Hero image loads quickly
- [ ] Lazy loading implemented for below-fold images
- [ ] Retina-ready images for high-resolution displays
- [ ] Images have proper dimensions (no oversized files)
- [ ] Background images optimized
- [ ] Product/jewelry images are high-quality and professional
- [ ] Images have appropriate alt text for accessibility

### Page Speed
- [ ] Overall page load time under 3 seconds
- [ ] First Contentful Paint (FCP) under 1.8 seconds
- [ ] Largest Contentful Paint (LCP) under 2.5 seconds
- [ ] Time to Interactive (TTI) under 3.8 seconds
- [ ] Cumulative Layout Shift (CLS) under 0.1
- [ ] CSS minified and combined where possible
- [ ] JavaScript minified and deferred/async loaded
- [ ] Unnecessary third-party scripts removed
- [ ] Browser caching configured
- [ ] CDN configured (if applicable)
- [ ] Gzip compression enabled
- [ ] Test with Google PageSpeed Insights (score 90+)
- [ ] Test with GTmetrix
- [ ] Test with WebPageTest.org

### Content & Copy
- [ ] Headline is clear and compelling
- [ ] Value proposition is immediately visible
- [ ] Benefits are clearly stated
- [ ] Social proof elements included (testimonials, reviews)
- [ ] Trust indicators visible (certifications, guarantees)
- [ ] No spelling or grammatical errors
- [ ] Scarcity/urgency elements (if applicable)
- [ ] Clear call-to-action buttons with action-oriented text
- [ ] Contact information easily accessible

- [ ] Review and optimize keywords
- [ ] Adjust ad copy based on performance
- [ ] A/B test results analysis
- [ ] Mobile vs desktop performance comparison
- [ ] Browser performance comparison
- [ ] Geographic performance analysis

- [ ] Monthly comprehensive analytics review
